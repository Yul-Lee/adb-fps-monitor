图表说明

右侧图表区域包含 6 个实时图表，共享 X 轴（时间），支持十字线联动和拖拽平移。

FPS 曲线

显示每秒帧数（FPS）的实时变化。曲线越平稳、越高，表示流畅度越好。

  • Jank 标记条：红色柱状标记，表示帧时间超过卡顿阈值的时间点
  • 平均线：黄色虚线，显示本次会话的平均 FPS

帧时间

每帧的渲染时间（毫秒），由 1000/FPS 计算得出。

  • 60 FPS → 16.7ms
  • 30 FPS → 33.3ms
  • 超过 33.3ms 的帧会被标记为 Jank

温度

设备各区域的实时温度（°C），数据来自 sysfs thermal_zone 传感器。

通过 ⚙ 设置面板可控制哪些温度传感器显示在图表中。

CPU/GPU 频率

CPU 各集群的运行频率（MHz）和 GPU 频率/负载。

  • CPU 频率来自 cpufreq sysfs
  • GPU 频率/负载来自平台特定路径（骁龙 kgsl / Mali devfreq）

单核 CPU 负载

每个 CPU 核心的使用率百分比，数据来自 /proc/stat。

单核 CPU 频率

每个 CPU 核心的当前运行频率（MHz），数据来自 cpufreq sysfs。

图表交互

  • 十字线联动：鼠标悬停在任意图表上时，其他图表会同步显示参考线
  • 拖拽平移：按住鼠标左右拖拽可平移时间轴
  • 底部导航条：拖拽蓝色选区可快速定位时间范围
  • 自动滚动：新数据到达时自动滚动到最新位置

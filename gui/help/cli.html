命令行参数

工具支持通过命令行参数控制启动行为：

参数说明默认值
-s, --serial指定设备序列号（多设备时使用）自动选择第一个
-p, --package指定目标应用包名自动检测前台应用
-i, --intervalFPS 采样间隔（秒，最低 1.0）1.0
--no-temp关闭温度监控开启
--no-freq关闭频率监控开启
--debug开启调试日志，输出到 adb_fps_debug.log关闭

使用示例

指定设备和包名：

python adb_fps_monitor.py -s DEVICE_SERIAL -p com.example.app

关闭温度监控，2 秒采样间隔：

python adb_fps_monitor.py --no-temp -i 2.0

开启调试日志：

python adb_fps_monitor.py --debug

环境要求

  • Python：3.10+
  • 依赖：PyQt6 >= 6.5.0、pyqtgraph >= 0.13.0、numpy >= 1.24.0
  • ADB：工具内置 platform-tools，也可使用系统 PATH 中的 ADB
  • 设备：Android 7+ 的设备，需开启 USB 调试

快速开始

1. 连接设备

通过 USB 数据线连接 Android 设备，并确保已开启 USB 调试（设置 → 开发者选项 → USB 调试）。

也支持 WiFi ADB 连接，但每次命令调用会有 100-200ms 额外延迟。

提示：连接后可在终端运行 adb devices 确认设备已识别。

2. 选择设备

启动后，左侧面板的设备下拉框会自动列出已连接的设备。如有多个设备，选择目标设备即可。

3. 开始监控

点击 ▶ 开始按钮，工具会自动检测当前前台应用并开始采集数据。

右侧图表区域会实时显示 FPS、温度、频率等数据曲线。

4. 暂停 / 继续

监控过程中可点击 ⏸ 暂停暂停数据采集（Worker 线程继续运行），再次点击 ▶ 继续恢复。

5. 结束监控

点击 ⏹ 结束停止监控。停止后按钮变为 💾 保存数据，可将本次监控数据导出为 CSV 文件。

6. 录制数据

监控过程中可随时点击 ⏺ 录制开始实时录制，录制的数据会在停止后保存。

设置面板

点击左侧面板的 ⚙ 设置按钮打开设置面板。设置面板是独立的浮动窗口，始终显示在最前方。

传感器选择

设置面板按类别分组显示所有已发现的传感器，通过复选框控制哪些传感器在图表中显示：

  • 🌡 温度传感器：控制温度图表中显示哪些温度曲线
  • ⚡ CPU/GPU 频率：控制频率图表中显示哪些频率曲线
  • 📊 单核 CPU 负载：控制单核负载图表中显示哪些核心
  • 📈 单核 CPU 频率：控制单核频率图表中显示哪些核心

默认勾选

以下温度传感器默认勾选（最重要）：

  • CPU、GPU、表面、电池
  • CPU大核、CPU中核、CPU小核

其他传感器默认可见但未勾选，按需手动开启。

传感器排序

传感器按类别自动排序：CPU → GPU → SoC → NPU → 电池 → 表面 → 内存 → WiFi → 相机 → PMIC → 充电 → 射频 → 其他。
